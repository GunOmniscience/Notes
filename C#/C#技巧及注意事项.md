# C#技巧及注意事项

## 1.C#类型的值的严格

C#在语法方面是一个特别严谨的一门语言。

例：我们在给float类型的变量赋值的时候我们需要注意它的值如果填3.0这是会报错的，因为我们的3.0默认是双精度的值。所以我们得在后面加F

float a = 3.0F; //float类型的值需要末尾加F来创建该类型的值，不然都是双精度的值会报错 double b = 3.0; 

## **2.C#中数据类型最多存放多少位数:**

int：10位

short：5位

long：19位

float：小数点后6位，如果有第七位则按四舍五入算；需要末尾加F来创建该类型的数据类型否则报错。

double：小数点后14位，如果有第15位则四舍五入算；数字本来就是双精度浮点型所以不需要像float类型那样。可加D没问题

## **3.如果我们发生运行时异常的时候:**

我们可以把运行时不异常的语句写在try里，然后把产生运行时异常的时候想显示的话写在catch里。

try {    Console.WriteLine("请输入:");    int a = Convert.ToInt32(Console.ReadLine());    Console.WriteLine(a); } catch (Exception e) //Exception是运行时异常的类，而e是我们自己取的变量我们通常以e来代表我们的Exception这个异常类 {    Console.WriteLine("输入错误！"); //此时如果我们运行时异常了则会打印“输入错误！” }finally{//是不管怎么样都要执行的方法体。我们可写可不写看情况。这是try的一个完整体系。     }
